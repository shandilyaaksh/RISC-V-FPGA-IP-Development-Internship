# RISC-V-FPGA-IP-Development-Internship
Documentation and lab work for VSD FPGA Internship Screening Program, including setup, RTL design exercises, and learning outcomes.

## Internship Objective

The objective of this task is to understand the complete RISC-V software development flow, including:

* Writing and executing a C program
* Compiling code using GCC
* Cross-compiling code using the RISC-V GCC toolchain
* Generating RISC-V executable files
* Analyzing RISC-V assembly instructions using objdump
* Comparing outputs obtained from the C model and the compiled RISC-V executable
